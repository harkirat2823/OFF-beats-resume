# OFF-beats-resume
This project, “Offbeats,” is a modern and interactive portfolio website created by a team of three students from Chandigarh University. It showcases their skills in web design and full-stack development through a visually engaging and highly responsive interface. The website features a clean navigation system, animated hero section, and well-structured sections such as About, Skills, Projects, and Contact.

Built using technologies like HTML, CSS, JavaScript, React, Django, and PostgreSQL, the project highlights both frontend creativity and backend functionality. It includes detailed project showcases like e-commerce platforms, task management tools, and data dashboards, demonstrating practical implementation of real-world applications.

The design focuses on user experience, smooth animations, and modern UI elements such as glassmorphism and neon effects. Overall, Offbeats represents a blend of creativity and technical expertise, aiming to build impactful digital experiences while continuously learning and improving as developers.
